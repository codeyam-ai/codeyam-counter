# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability, please report it privately. **Do not
open a public issue for security problems.**

Email **security@codeyam.com** with:

- A description of the vulnerability and its potential impact
- Steps to reproduce (proof-of-concept if possible)
- Any suggested remediation

You can expect an acknowledgement within a few business days. We'll keep you
informed as we work on a fix and will credit you in the release notes unless
you prefer to remain anonymous.

## Supported versions

This project is released from `main`. Security fixes are applied to the latest
release.
