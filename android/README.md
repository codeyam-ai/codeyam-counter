# Kotlin + Jetpack Compose (Android) Tech Stack

A minimal-but-real native Android app scaffold using Jetpack Compose, Material 3, and Gradle Kotlin DSL.

## Getting Started

This template contains a single-module Gradle project:
- **`app/`**: The core Android application module.
- **`build.gradle.kts`**: Root build configuration.
- **`settings.gradle.kts`**: Root settings listing the `:app` module.

Refer to [MOBILE_SETUP.md](MOBILE_SETUP.md) for pre-flight and simulator testing guides.
